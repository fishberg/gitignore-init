# gitignore-init

A command-line tool that helps you copy a `.gitignore` from the official GitHub repo:
- https://github.com/github/gitignore

## Install
Install cloned repo:
```
pip install -e .
```

Install directly from GitHub:
```
pip install git+https://github.com/fishberg/gitignore-init
```

## Usage
```
gitignore-init
```
